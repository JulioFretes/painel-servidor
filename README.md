# painel-servidor
Painel pra administrar o servidor de minecraft

```
npm run install:all
npm run dev
```

Pra rodar indepedentemente usar:
```
npm run start:front-end
```
ou
```
npm run start:back-end
```

Para compilar no linux
```
rm -r /var/www/html/dist
npm run serve
```
